# Diabetic Retinopathy Exudate Segmentation

This repository contains code related to exudate segmentation in retinal fundus images for diabetic retinopathy detection.

The project is connected to the paper **"Exudates Segmentation in Retinal Fundus Images for the Detection of Diabetic Retinopathy"** by Soumil Chugh, Jaskirat Kaur, and Deepti Mittal.

## Project Context

Diabetic retinopathy screening often relies on identifying visible retinal abnormalities in fundus images. Exudates are one such marker, and this project explores image-processing methods for segmenting exudate regions from retinal fundus imagery.

## Focus Areas

- Retinal fundus image analysis
- Exudate segmentation
- Medical image preprocessing
- Classical computer-vision methods for diabetic retinopathy screening
